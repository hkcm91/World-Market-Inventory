Template

Print at 100% / Actual Size with "Fit to page" OFF — anything less shrinks the sheet and the rows drift out of the labels. Each card is one product; Copies (the ×N badge) is how many of that label print — duplicates fold into one card automatically. Qty/box prints on the label; set 0 for no number. Full description prints on its own line and auto-fits so it never gets cut off. Set a date to print a third line and to include it in Move dated → Expiry. Switch to the Photo template to print a product image on each label — photos load automatically for items World Market carries online; add your own for the rest with Add photo. Tap 🎤 Say date to speak an expiry (e.g. “August twelfth twenty twenty six”). Check the labels you want, hit Download, then Mark printed — it takes only the checked ones, moves them to the Archive tab and drops them off this list, so this list always shows just what’s left to print. Restore anything from the Archive tab whenever you need it again. Set Behind to how many boxes of that SKU sit behind the front one and it auto-adds “1 BEHIND”, “2 BEHIND”… stock-depth labels after it. SKU info (name, qty/box, photo) is remembered, so re-adding a SKU auto-fills. Backup saves everything to a file; Restore brings it back. Everything saves on this device.

Scan to import

Pick a workflow, then scan a barcode, or point at a box, shelf tag, or pick-ticket label (even sideways) and tap ✨ Read box — it reads the text and guesses the SKU, name, qty and expiry to pre-fill for you (review before importing). Doing a stack? Tap ▶️ Auto-read and just hold the camera over each ticket — it reads them hands-free and beeps as each new SKU lands. Review the list, then ✓ Approve all → Archive sends the whole batch to the Archive (photos fill in automatically). No camera? Type or paste a code below.
